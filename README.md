# CodeRealm 🌍

CodeRealm is an experimental 3D world for exploring Data Structures and Algorithms in a more visual and interactive way.

The project is currently in its early stages. The initial goal is to build a browser-based 3D world where different DSA topics can eventually be represented as unique regions, with coding problems arranged as interactive nodes.

## Current Progress

- Set up React with Vite
- Integrated Three.js using React Three Fiber
- Added interactive camera controls with Drei
- Added basic scene lighting
- Built the first low-poly floating island prototype

## Planned Direction

- Create distinct regions for Arrays, Trees, Graphs, and Dynamic Programming
- Add interactive topic regions
- Represent coding problems as spatial 3D nodes
- Add problem paths and progression
- Experiment with XP and gamification

## Tech Stack

- React
- Vite
- Three.js
- React Three Fiber
- Drei

## Project Status
